HTML e CSS

Informações básicas:

Apenas noções iniciais de HTML e CSS para iniciantes.

    Estrutura HTML:

  1. A tag "!DOCTYPE html" é usada no início do arquivo e sinaliza ao navegador qual versão do html está sendo usada.
  2. A tag "html" marca o conteúdo que será renderizado pelo navegador. É acompanhada da propriedade "lang", indicando o idioma da página.
  3. A tag "head" é usada no início e no fim das informações que estamos passando para o navegador.
  4. Para que os caracteres e acentuação correta sejam exibidos, usamos a propriedade "charset", acompanhada do dicionário desejado, dentro da tag "meta".
  5. Integramos HTML e CSS por meio da tag "link", referenciando o arquivo externo do CSS.
  6. A tag "body" é usada no início e no fim do conteúdo que queremos exibir na página.

    Estrutura CSS:

  1. O CSS pode ser integrado como arquivo externo, inline (dentro de outras tags) ou dentro do "head", na forma da tag "style".
  2. Editamos elementos nomeando-os e abrindo chaves com o conteúdo que queremos editar.
  3. Referenciamos classe por meio de um ponto ".", e id por meio de uma hashtag "#".
